# 🎂  Smart Age Calculator

A modern and fully functional **Age Calculator Web App** built using **HTML, CSS, and JavaScript**.  
Users can enter their birth date and instantly calculate their exact age in **Years, Months, and Days**.

## ✨ Features

### Core Features
- Input fields for Day / Month / Year
- Calculates exact age in Years, Months, Days
- Smooth animated number counter
- Responsive modern UI

### Validation Features
- Checks empty inputs
- Validates month (1-12)
- Validates correct day based on month
- Leap year support
- Rejects future dates
- Highlights invalid fields in red

### Bonus Features
- Total days lived
- Total weeks lived
- Total months lived
- Countdown to next birthday

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📚 Key Concepts Used
- DOM Manipulation
- JavaScript Date Object
- Form Validation
- CSS Animations
- Responsive Design

2. Open `index.html` in browser
